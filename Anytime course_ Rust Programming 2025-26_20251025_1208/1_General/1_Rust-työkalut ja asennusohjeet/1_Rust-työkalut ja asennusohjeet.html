Rust

Voit käyttää Rust-ohjelmointikieltä Windows, Mac tai Linux laitteeltasi. Asennus tapahtuu rustup työkalun kautta.

Asentaaksesi Rustin, rustup:n, avaa linkki: https://www.rust-lang.org/tools/install


Windows-ohjeet

Windows laitteelle, lataa käyttäen installeria:

Installer löytyy 32 ja 64-bittiselle tietokoneelle. Jos tietokoneesi on vain muutaman vuoden vanha, on se todennäköisesti 64-bittinen, jolloin kannattaa ladata 64-bittinen versio, sillä se on hieman nopeampi.

Linux-ohjeet

Linkki Unix-version asennuskomentoon löytyy täältä: https://forge.rust-lang.org/infra/other-installation-methods.html#other-ways-to-install-rustup.

Komento:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

MacOS-ohjeet:
MacOs käyttää samaa komentoa kuin Linux: https://forge.rust-lang.org/infra/other-installation-methods.html#other-ways-to-install-rustup.

Komento: 

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Version tarkistus

Kun olet asentanut Rustin, avaa komentokehote ja aja komento:

rustup --version
rustc --version


Jos asennus onnistui, pitäisi tulostuksen olla kuvankaltainen, versioluvun tulostua sekä rustup-komennolla ja rustc-komennolla.


Komentokehotteen avaaminen:

Windows:
Windows + R -> "cmd" -> paina enter


VSCode:

Voit avata komentokehotteen VSCoden sisällä kuvassa näkyvällä tavalla.


Komentokehote avautuu alakulmaan. Windowsissa komentokehote on yleensä joko PowerShell tai Command Prompt.

Hyödyllinen Rust-lisäosa VSCodelle

VSCodelle on saatavilla hyödyllinen lisäosa rust-analyzer, joka nopeuttaa dokumentaation löytämistä sekä virheiden korjaamista antamalla hyödyllistä tietoa jo ennen ohjelman kääntämistä.
Voit asentaa rust-analyzerin myös muille ohjelmointiympäristöille - tarkemmat ohjeet löytyy rust-analyzerin dokumentointi sivulta kohdasta Installation.
Alla olevasta kuvasta näemme miten rust-analyzerin voi asentaa VSCodeen.
Vasemmalta sivupalkista valitsemme extensions, haemme rust-analyzer, ja painamme oikealle avautavasta ikkunasta "Install". Sivulta löytyy linkit configurointiin, dokumentointisivulle sekä lista ominaisuuksista.


Rust-analyzerin voi helposti konfiguroida lisäämällä esimerkiksi viikkotehtäviä varten luomaasi kansioon tiedosto "{kansion_nimi}.code-workspace" ja lisätä sinne kaikki kansiot, jotka sisältävät cargo-työkalulla luotuja Rust-projekteja. Esimerkkikuvassa alla on kaksi viikkotehtäväkansion alikansiota, joissa molemmissa on yksittäisiä cargo-projekteja jotka rust-analyzer osaa tunnistaa.


Kun olet luonut uuden projektin cargo new komennolla ja rust-analyzer on asennettuna, pitäisi VSCoden antaa suorittaa ohjelma nappia painamalla main-funktion yläpuolelta (katso alla oleva kuva), sekä antaa tietoa eri komennoista silloin, kun pidät hiirtä esimeriksi tunnetun makron tai standardikirjaston funktion, metodin tai makron kohdalla. Kun ajat ylläolevan ohjelman, avautuu rust-analyzerin komentokehote, joka antaa suorituksen sekä pysyy auki, kunnes suljet terminaalin painamalla mitä vain näppäintä.


Huom!

Jos ohjelmasi vaatii komentoriviargumenttien käyttöä, pitää sinun ajaa ohjelmasi cargo run -komennolla jotta voit ajaa ohjelman antamiesi argumenttien kera.
